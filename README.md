📝 Description

This project is my first task as part of the Tutedude Learning Program.
It contains a simple static web page created using plain HTML without any external libraries, frameworks, or CSS.

The purpose of this task is to understand the basic structure of an HTML document and learn how different HTML tags work together to create a web page.

🚀 Features

Uses only pure HTML

No CSS or JavaScript

Beginner-friendly structure

Clean and readable code

Easy to understand for new learners

🛠️ Technologies Used

HTML5
